3.功能

报表栏位说明

序 栏位名称 表 字段 类型 大小 栏位说明
1 成本对象 bd_material name VARCHAR2 300 金属产品 如：金属镧
2 泰国产量 pcto_srbill_b yield 数量 Sum(收入单据实际产量)
3 电解车间 pcto_costbillfromfentan sum NUMBER 28 Sum泰国公司成本单据（分摊）成本项目名称=制造费用xxx的成本金额

5 销售部门费用 pcto_costbillfromfentan sum NUMBER 28 Sum泰国公司成本单据（分摊）成本项目名称=销售费用xxx的成本金额
6 财务部费用 pcto_costbillfromfentan sum NUMBER 28 Sum泰国公司成本单据（分摊）成本项目名称=财务费用XXX的成本金额
7 氟化锂金额 fhlje FHLMONEY NUMBER 28 统计盐里面含有的氟化锂的成本金额
11 合计1 fc 表达式 数字 统计总成本 电解车间+管理部费用+销售部+财务部+氟化锂金额
12 单位成本1 fc 表达式 数字 计算得出单位成本 (合计1/泰国产量)

14 盐中氟化物金额 N/A expression 序13 X 序17
15 氟化物耗用量 tgscbcbft_02 NNUM 数量 取泰国的氟化物消耗量
16 氟化物金额 N/A expression 序15 X 序17
17 氟化物单位成本 N/A expression 如果产品原料是金属镧就取市场价，否则取科力的单位成本，取不到就取氟化物的标准单价。
18 合计 fc 表达式 数字 电解车间+管理部费用+销售部+财务部+氟化锂金额+盐中氟化物金额+氟化物金额
19 单位成本 fc 表达式 数字 合计/泰国产量=单位成本
  • 盐中氟化物金额 = 盐中氟化物数量 X 氟化物单价

sum(nvl(tgscbcbft_08.NNUM,0) * case when tgscbcbft_02.MATNAME=’金属镧’ then t_5.MINPRICE else nvl(t_4.DWCB,nvl(t_3.UNIT_PRICE,0)) end )

    1、请确保node版本大于6.2
    2、在博客根目录（注意不是yilia根目录）执行以下命令：
    npm i hexo-generator-json-content --save

    3、在根目录_config.yml里添加配置： 

      jsonContent:
    meta: false
    pages: false
    posts:
      title: true
      date: true
      path: true
      text: false
      raw: false
      content: false
      slug: false
      updated: false
      comments: false
      link: false
      permalink: false
      excerpt: false
      categories: false
      tags: true
